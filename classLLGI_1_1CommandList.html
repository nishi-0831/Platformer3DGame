詳解

command list

覚え書き
CommandList has a swap buffer. So you don't need to consider buffering. CommandList has references related to rendering. References are released after the number of swapCount has passed.

Limitation : Begin and End are need to call only once in one frame. Command list must not be released after finishing rendering.
